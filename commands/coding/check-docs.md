# Check Documentation Accuracy

Scan all documentation files and verify they match the current code:

1. Find all `.md` files (especially `README.md` and `docs/`)
2. Check that:
   - Referenced files/modules exist
   - Code examples are correct
   - Installation instructions match requirements
   - CLI arguments match actual code
   - API descriptions are accurate
   - Outputs match actual behavior

Report any discrepancies found.
